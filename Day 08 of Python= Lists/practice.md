🟢 Basic Level

1. Create a list of 5 numbers and print it.
2. Print the first and last element of a list.
3. Replace the second element of a list with a new value.
4. Add an element at the end and at a specific position.
5. Remove a specific element and the last element.

🟡 Intermediate Level

6. Find the maximum and minimum in a list.
7. Find the sum of all elements in a list.
8. Count even and odd numbers in a list.
9. Reverse a list without using built-in functions.
10. Check if an element exists in a list.

🔵 Advanced Level

11. Remove duplicate elements from a list.
12. Find the second largest number in a list.
13. Merge two lists into one.
14. Create a list of squares using list comprehension.
15. Count frequency of each element in a list.

🔴 Challenge Level

16. Check if a list is a palindrome.
17. Rotate a list to the right by one position.
18. Flatten a nested list.
19. Find common elements between two lists.
20. Split a list into even and odd lists.
