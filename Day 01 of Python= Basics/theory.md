Python

It is a high-level , interpreted programming language known for:
•	Simple & readable syntax 
•	Platform independence
•	Wide use in AI, Web development, Data science, Automation

It was created by Guido Van Rossum (1991).

Features of Python
•	Easy to learn (beginner-friendly)
•	Interpreted (no compilation needed)
•	Object-Oriented 
•	Large libraries ( Numpy, Pandas, etc.)
•	Cross-platform

Installing Python
Steps:
1.	Go to official website: https://python.org
2.	Download latest version
3.	Install and check  “Add Python to PATH”
Verify Installation: 
      python –version

First Python Program

print(“ Hello World”)
 
Explanation:
•	print() -> used to display output
•	Text inside quotes -> string

Python Syntax Basics
•	No semicolon required
print(“Hello!”)
print(“World”)
•	Indentation Matters

Indentation means giving spaces at the beginning of a line to define a block of code. In python, indentation is mandatory (unlike C, C++, Java where{} are used).

Code => if True:
                       print(“Correct Indentation”)

Comments in Python 
# This is a comment
print("Hello World")          # This prints output

Variables in Python
name = "Devanshi"
age = 20

Rules:
•	No need to declare type
•	Dynamic typing

Basic Data Types
a = 10        # int
b = 3.14      # float
c = "Hello"   # string
d = True      # Boolean

Taking Input
name = input("Enter your name: ")
print("Hello", name)
Input is always taken as string by default.

Type Casting
a = int(input("Enter number: "))
b = float(input("Enter decimal: "))

Basic Operators
a = 10
b = 3
print(a + b)   # Addition
print(a - b)   # Subtraction
print(a * b)   # Multiplication
print(a / b)   # Division
Simple Program Example
# Program to add two numbers

a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
sum = a + b
