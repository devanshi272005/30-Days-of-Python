🔷 Practice Questions on Functions


🟢 Basic Level

* Write a function to print "Hello World"
* Write a function to find the sum of two numbers
* Write a function to check whether a number is even or odd
* Write a function to find the square of a number
* Write a function to print numbers from 1 to n

🟡 Intermediate Level

* Write a function to find the maximum of three numbers
* Write a function to calculate the factorial of a number
* Write a function to check whether a number is prime or not
* Write a function that takes a list and returns the largest element
* Write a function to reverse a string

🟠 Advanced Level

* Write a function using *args to find the sum of all numbers
* Write a function using **kwargs to print student details
* Write a recursive function to find Fibonacci series
* Write a lambda function to find cube of a number
* Write a function to count vowels in a string
