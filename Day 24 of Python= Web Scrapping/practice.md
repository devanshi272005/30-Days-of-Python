🔹 Practice Questions (Web Scraping)

🔸 Basic Level

Write a program to fetch a webpage and print its title.
Extract all <p> tags from a webpage and print their text.
Extract all <h1> headings from a webpage.
Print all links (href) present on a webpage.
Count the total number of paragraphs in a webpage.

🔸 Intermediate Level

Extract all headings (h1, h2, h3) from a webpage.
Print only those links that start with https.
Extract text from a specific class (e.g., "content").
Scrape all items from a page and print their names and prices.
Store all extracted data into a text file.

🔸 Advanced Level

Scrape data from multiple pages (pagination).
Handle errors like connection issues or invalid URLs.
Extract attributes like src from <img> tags.
Clean and format scraped data before printing.
Create a reusable function for scraping any webpage.
